## 提交前建议核对

1. 打开 GitHub 仓库，确认最新 README 为中文增强版。
2. 打开 mooncakes.io 包页面，确认版本和文档可访问。
3. GitLink 页面同步 GitHub 仓库后，确认仓库公开可访问。
4. 在作品提交页填写作品名称、作品链接、包链接和简介。
5. 上传开发报告或源码包。
6. 用户本人检查个人信息和最终提交按钮。
