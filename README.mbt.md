# MoonBench

MoonBench is a lightweight stopwatch, benchmark reporting, and baseline
comparison toolkit for MoonBit projects.

It is built for the 2026 MoonBit open-source ecosystem contest. The project
focuses on a practical package that MoonBit developers can use to measure small
functions, export stable reports, and compare current results with a baseline.

## Planned capabilities

- Stopwatch helpers for manual measurements.
- Benchmark runner built on MoonBit's core timing facilities.
- Summary statistics for measured samples.
- JSON and Markdown reports.
- Baseline comparison for performance regression checks.

## Development

```powershell
moon check
moon test
moon run cmd/main
```
