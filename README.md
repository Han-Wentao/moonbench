# MoonBench

MoonBench is a lightweight stopwatch, benchmark reporting, and baseline
comparison toolkit for MoonBit projects.

## Status

- MoonBit package: in development for OSC2026 MoonBit ecosystem contest.
- Tests: `moon test`
- Demo: `moon run cmd/main`
- License: MIT

See [README.mbt.md](README.mbt.md) for full package documentation.
